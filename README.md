
# Course Registration

This is made for registration of a technical institution who provides services on many technical courses.
## Three Features
- Anyone can control total credit and remaining credit in a same time using select button.
- Anyone can reuse this projects codes for future .
- Course registration is an easy and readable project which is very much easy to understand all codes.
